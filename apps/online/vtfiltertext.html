VT Filter (text)

This app filters the text from all the VT sequences, including the CSI sequences, OSC sequences, and many other supported sequences. This is useful for text that contains these sequences so that the filtered text can be used for printing to consoles that don't support these sequences, localizing them, and so on.

Click on the Filter button to remove the VT sequences from the input text. If you want the Python version of the filter, click on the Python icon.
