ReadLine.Reboot

ReadLine.Reboot was created as a reboot to the original ReadLine library made by Toni Solarin-Sodara, which was abandoned. It is a port of GNU Readline to .NET.

VisualCard

VisualCard is a vCard parser that is made entirely in .NET with no dependencies. It supports vCard versions 2.1, 3.0, and 4.0 and vCard files containing one or more contacts. It is currently under development.
