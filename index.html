Welcome to APTIVI!

...where tech changes

We, Aptivi, were established in 2014 as an initial PC repair and expertise center to save the world. Over time, we are getting expanded to programming, so we have made some of the projects that we were proud of. We are not only interested in PCs, but also interested in technology in general, which is our main role.

Top Projects

Kernel Simulator

Kernel Simulator simulates our future-planned computer kernel that we are building. It is made entirely in .NET with multiplatform support.

  • Version

ReadLine.Reboot

ReadLine.Reboot was created as a reboot to the original ReadLine library made by Toni Solarin-Sodara, which was abandoned. It is a port of GNU Readline to .NET.

  • Version

GRILO

GRILO is a managed bootloader simulator made entirely with .NET. It allows you to "boot" to .NET assemblies that are compatible with GRILO. You too! You can make your .NET application bootable!

  • Version

Our favorite projects

Newtonsoft.Json

Newtonsoft.Json is a godlike JSON file parser, maker, and manipulator that allows you to interact with JSON files, create them, edit their properties, add them, and much more. It is one of the most downloaded NuGet packages, and we strive to use it to our projects that use JSON, like Kernel Simulator and GRILO. Newtonsoft.Json is made by James Newton-King.
