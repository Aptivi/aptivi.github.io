Welcome to APTIVI!

...where tech changes

We, Aptivi, were established in 2014 as an initial PC repair and expertise center to save the world. Over time, we are getting expanded to programming, so we have made some of the projects that we were proud of. We are not only interested in PCs, but also interested in technology in general, which is our main role.

Top Projects

Kernel Simulator

Kernel Simulator simulates our future-planned computer kernel that we are building. It is made entirely in .NET with multiplatform support.

ReadLine.Reboot

ReadLine.Reboot was created as a reboot to the original ReadLine library made by Toni Solarin-Sodara, which was abandoned. It is a port of GNU Readline to .NET.

  • Version

VisualCard

VisualCard is a vCard parser that is made entirely in .NET with no dependencies. It supports vCard versions 2.1, 3.0, and 4.0 and vCard files containing one or more contacts.
